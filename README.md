# conventional-fizzbuzz

이 프로젝트는 fizzbuzz와 git의 conventional commit 을 연습하는 저장소입니다.

[See Demo](https://www.googlt.com/)

## Prerequisites

 - python==3.11.0
 - fastapi
 - jupyter

## How to start
```shell
$ git clone {username/repo addr}
$ cd {repo addr}

$ python fizzbuzz.py
```

## Installation
```shell
$ poetry install
or
$ pip install -r requirements.txt
```

## features

 - Print fizzbuzz
 - Print fizz
 - Print buzz
